# tools/

本目录存放**一次性临时调试脚本**，不是项目正式组件。

## 来源

这些脚本原本放在项目根目录，是开发过程中用于排查 `index.html` 数据注入问题的临时工具（例如统计 `胜其他/平其他/负其他` 字段、检查比分区块数量等）。已统一移入本目录，并把硬编码的本地 Windows 路径（`C:\Users\Jin\Desktop\...`）改为相对路径。

## 使用

```bash
# 从项目根目录执行（脚本内部用 __file__ 自动定位）
python tools/check_scores.py
python tools/fix_other.py     # 注意：会直接修改 ../index.html
```

## 注意

- 大多数脚本**只读**分析，可放心运行
- `fix_*.py` 会**修改** `index.html`，运行前建议先 `git commit` 当前状态以便回滚
- 这些脚本对应 v7.5x 版本的清理逻辑，后续如果结构有变可能失效，失效后请直接删除

## 是否应该保留？

可以评估后决定：
- **保留**：未来如果再次出现数据格式问题可参考
- **删除**：如果 git 历史里有，删了也不心疼

不需要时直接 `git rm tools/*.py` 即可。
